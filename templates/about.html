About the Project

This project is an AI-powered Emotion Detection System that uses a deep learning image segmentation model to identify whether a face is happy 😊 or sad 😢.

Built using TensorFlow, Flask, and modern HTML/CSS/JS, this web application is designed to provide a fast and interactive prediction experience.

You can upload any image of a face, and our model will smartly predict the emotional tone of the face in just a second.
